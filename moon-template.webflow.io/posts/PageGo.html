A Website Programming language

Go is a general purpose programming language with advanced features and a clean syntax. Because of its wide availability on a variety of platforms, its robust well-documented common library, and its focus on good software engineering principles, Go is an ideal language to learn as your first programming language.

History

  • Go is statically typed, compiled programming language which was designed at Google in 2007 and it was publically announced in November 2009.
  • The Go design was primarily motivated by their shared dislike of C++ language.
  • After some years in March 2012 its version 1.0 was released. And after that they released many version in some years.
  • The latest version of Go is 1.14 which was released on 25th of February 2020.
  • The main developers of Go language is Robert Griesemer, Rob Pike, Ken Thompson.

Overview & features

  • Go have the same syntax as we uses in C Language.
  • Go language readability and usability is same as the Python and JavaScript.
This language mainly developed to improve programming productivity in an era of multicore, network machines and large codebases.
  • But Go have some extra features such as memory safety, garbage collection, structural typing and concurrency.
  • The language is mainly known as “Golang” due to his domain name, golang.org but its proper name is GO.

Printing text & data in Go

  • Go language is same as C language there is minor changes in printing a statement.

Comments

  • To give single line comments we have to use “//”.
  • Where as to give multiline comment we have to use “/*” to start the comment section and “*/” to end the comment section.
  • The syntax is same as the most of the other programming languages.

Benefits of Go

  1. Ease of use

    Go is easy to understand. The syntax is clean and accessible to newcomers and there aren’t any complex structure to learn. The close similarity to C language, it will be really easy to learn for C++ or C# proficient.

  2. Smart Standard Library

    Go users can accomplish a lot without having to import or learn complicated secondary libraries. The standard library that comes packaged with Go is sophisticated without overwhelming, and it reduces the risk of errors from conflicting function names.

  3. Strong Security Built-in

    In Go there is a built in garbage collector which helps to prevent memory from bleeding off in your code.

Drawbacks of Go

  1. Too Simplistic

    One of the Go main advantage is also one of the biggest weakness. Go may be an easy language to pick up, but that brings with lack of versatility.

  2. Lack of Virtual Machine

    Most of the languages are now based on virtual machines so they are more efficient but Go is not based VMS which is Weakness of it.

  3. Still Young Language

    There is a lot of promise by Go, but it’s a language which still in process that means it can’t compete with similar languages. Go native library may be designed smartly and efficiently, but it competing with languages like Java that comes supported by huge collection of code.
