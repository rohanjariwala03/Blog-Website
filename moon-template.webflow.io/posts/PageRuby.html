A Website Programming language

Ruby is a powerful, flexible programming language you can use in web/Internet development, to process text, to create games, and as part of the popular Ruby on Rails web framework. The # sign is for comments in Ruby. A comment is a bit of text that Ruby won’t try to run as code.

History

  • Ruby is the general purpose programming language. .
  • The first version of ruby, 0.95 was published domestically in japan on 21st December 1995.
  • Ruby was invented by a Japanese computer scientist Yukihiro "Matz" Matsumoto in the decade of 1980 to 1990.
  • This language named “Ruby” before writing the base code of the language.
  • Ruby achieved massive acceptance in year 2006.
  • The primary purpose of Ruby was to help programmers to be creative and to enjoy the programming.
  • Ruby was developed because Yukihiro wants feasible scripting and truly object-oriented language.

Overview & features

  • Ruby is server-side scripting language and it is similar to Python and PERL.
  • Ruby is one of the open source programming languages.
  • Ruby has two statements: “BEGIN” and “END”. BEGIN block executes first and after that other statements are executed, and at last END statement is executed.
  • We can use it without paying any cost.
  • It can be embedded into Hyper Text Markup Language.
It is a language which can be easily and quickly learn by anyone because it has very clean and simple syntax. 
  • Ruby is that much flexible that we can maintain big programs smoothly.
  • It has plenty of predefined functions, so it can be directly used into scrips of ruby.
  • We can use Ruby for internet and intranet applications.
  • Ruby support numerous GUI tools like Tcl/Tk, GTK and OpenGL.
  • A ruby file have an extension of “.rb”.

Printing text & data in Ruby

  • We can display string and data by using “puts” keyword. It work same as the printf statement.
  • We can also use print statement as an alternative of puts statement.
  • Example:- puts “Bonjour”

Taking input in Ruby

  • “gets.chomp” is used to take input from user in Ruby.
  • We can also use only “gets” to get the string input.
  • The “gets” returns a string and a ‘\n’ character, while chomp removes ‘\n’ character.

Comments

  • For single line comment we have “#” in ruby.
  • For multiline comment “=begin” and “=end” is used.
  • Example:
    • # Hello   - This is single line comment.
    • =begin
      Hello,
      This is
      Multiline comment.
      =end

Variable Declaration and Scopes

  • The key feature of Ruby is we need not have to mention the type of variable at the time of variable declaration. We only have to write name of the variable and assign value to it. Ruby will automatically detect the type of variable.
  • In Ruby the identifiers are case-sensitive, so Str and str both are different identifiers.
  • Also it’s not necessary to define scope of the variable, Ruby automatically determines it.
  • To declare variable as local variable, we don’t have to use any operator.
  • To declare variable as global variable, we have to use “$” operator with the name of variable.
  • “@” operator is used to declare variable as an instance variable.
  • “@@x” operator is used to declare variable as a class variable.
  • Example :
    • n=100
    • $Str=”GlobalVariable”
